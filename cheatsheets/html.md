# Cheatsheet HTML

## Títulos

**Tag**:

```html
<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
<h4>Título 4</h4>
<h5>Título 5</h5>
<h6>Título 6</h6>
```

**Exemplo**:

<div style="background: #eee; padding: 7px;">
    <h1>Título 1</h1>
    <h2>Título 2</h2>
    <h3>Título 3</h3>
    <h4>Título 4</h4>
    <h5>Título 5</h5>
    <h6>Título 6</h6>
</div>

---

## Paragrafo

**Tag**:

```html
<p>Um exemplo de parágrafo.</p>
```

**Exemplo**:

<div style="background: #eee; padding: 7px;">
    <p>Um exemplo de parágrafo.</p>
</div>

---

## Listas ordenadas e não ordenadas

**Tag**:

```html
<ul>
    <li>Banana</li>
    <li>Maçã</li>
    <li>Uva</li>
</ul>
```

**Exemplo**:

<div style="background: #eee; padding: 7px;">
    <ul>
        <li>Banana</li>
        <li>Maçã</li>
        <li>Uva</li>
    </ul>
</div>
